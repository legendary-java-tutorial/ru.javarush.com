Для тех, кто всё пропустил: Условия конкурса Дайджест статей студентов JavaRush #1 за 17-19 февраля

А вот и новые статьи участников конкурса:

1. Создание Telegram-бота на Java: от идеи до деплоя

Студент JavaRush с никнеймом John Watson (27 уровень) написал очень полезную практическую статью о создании бота для Telegram. Автор не только расписывает как создать такое приложение, но где его развернуть, а также приводит ссылки на нужные для понимания определения и API для разработки.

2. MVP в Android для самых маленьких

Применять правильные практики и шаблоны программирования нужно не только в «Энтерпрайзе», но и в мобильной разработке. В этом уверен Алексей Дмитревский (31 уровень, Москва), который демонстрирует, как можно применить принцип MVP (Model-View-Presenter) при написании несложного Android-приложения с базой данных. Автор также приводит ссылку на GitHub с полным кодом разобранного примера.

3. Одно решение, изменившее жизнь

Igor Melnitsky (25 уровень JavaRush) начал учёбу на нашем курсе с вполне логичным намерением стать программистом, а ещё — потренировать мозг. Автор этой статьи считает, что именно тренировка мышления привела его к успешному бизнес-проекту, которым он сейчас занимается. А ещё дает несколько советов «новеньким».

4. Преимущества использования Spring

О Spring многие слышали, но те, кто только начинают свой путь «джависта», в основном не понимают, что это за фреймворк такой, и воспринимают его, как некую «страшилку», с которой им доведётся столкнуться в ближайшем будущем. Студент JavaRush с ником Alex (37 уровень) рассказывает новичкам о том, почему Spring — это хорошо.

5. Топ-7 книг по Java для начинающих

Max Stern (35 уровень) из Нижнего Новгорода делится списком полезных книг по Java, которые помогли ему разобраться с разными аспектами нашего с вами любимого языка во время учёбы. Автор рассказывает и о самых простых учебниках «с нуля», и о подробных справочниках, и о тех книгах, которые стоит прочитать каждому «джависту» для формирования правильного программистского мышления.
