Всплывающие подсказки (окна)
Всплывающие подсказки (окна) полезны для проверки дополнительной информации относящейся к элементу на котором установлен курсор. Приведем список всплывающих подсказок, которые помогут вам повысить производительность труда.
  1. Информация о параметре Информация о параметрах, необходимых для конкретного метода доступна по нажатию Ctrl + P ( Cmd + P ). Всплывающее окно показывает подходящий перегруженный метод, выделяя текущий параметр.
    • Быстрое описание Если Вы хотите быстро посмотреть на реализацию метода на котором установлен курсор вы можете вызвать быстрое описание по нажатию Shift + Ctrl + I (Shift + Cmd + I)
      • Документация Всплывающее окно с документацией является одним из самых часто используемых Чтобы его открыть, нажмите Ctrl + Q (Ctrl + J на Mac).
        • "Неисчезающая" подсказка (окно) Если вы уже открыли документацию и хотите оставить документацию открытой, кликните на кнопку Pin в верхнем правом углу. Будет показана (и не исчезнет) документация по текущему элементу.
          • Окна, всплывающие из панели Если вас не устраивают “плавающие” окна, вы можете подсоединить их к краю главного окна, нажав на кнопку в правом верхнем углу.
            • Всплывающие окна в других компонентах среды разработки Хорошо известно что всплывающие окна с подсказками могут быть вызваны для любого элемента кода в вашем редакторе. Однако, также они доступны и для элементов в любом другом списке, при использовании таких же сочетаний клавиш.
                Вы можете вызывать всплывающие подсказки даже когда перебираете элементы на панели навигации по нажатию Alt + Home (Alt + ↖ для Mac).
                • “Показать места использования” “Показать места использования” это очень простая, но мощная функция. Просто нажмите Alt + Ctrl + F7 (Alt + Cmd + F7 для Mac) и вы увидите список тех мест в вашем коде, где используется элемент (метод или объект) находящийся под курсором.
