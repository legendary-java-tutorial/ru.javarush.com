Друзья! Те из вас, кто уже успели установить IDEA 2018.1 и только затем попытались обновить плагин JavaRush (до версии 3.17), могли столкнуться со следующей проблемой: старая версия плагина была удалена, а новая так и не появилась после перезагрузки IDE. Дело в том, что в IDEA 2018.1 изменился метод обновления плагинов. Благо, эта проблема легко решается: можно установить плагин вручную.
  1. Зайти на JavaRush на страницу загрузки плагина и загрузить его
  2. Зайти в IntelliJ IDEA -> Открыть проект JavaRushTasks
  3. Зайти в File-> Settings (Alt+ Ctrl + S)
  4. В меню слева выбрать пункт Plugins
  5. Нажать на Install from Disc
  6. Выбрать загруженный плагин -> OK
После этого IDEA предложит перезагрузиться. Теперь плагин должен появиться и корректно работать.
