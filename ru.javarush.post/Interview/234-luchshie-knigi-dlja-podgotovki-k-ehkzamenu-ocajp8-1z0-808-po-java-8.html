Предлагаем вашему вниманию советы по подготовке к сертификации, опубликованные Software Architect компании Infotech Джейвином Полом.
Прошло уже больше трёх лет с 17-го марта 2014 г, появления Java SE 8 и уже почти три года все желающие могут сдать сертификационный экзамен OCAJP8. Наверное, вы в курсе: чтобы стать сертифицированным разработчиком Java SE 8, необходимо сдать два экзамена – OCAJP8 (код экзамена 1Z0-808) и OCPJP8 (код экзамена 1Z0-809). Первый из них называется сертификацией базового уровня (associate level certification), а второй – сертификацией профессионального уровня (professional level certification). Когда OCAJP8 только появился, не существовало достаточно хороших книг, посвященных подготовке к нему, так что многим кандидатам приходилось полагаться в основном на книги по Java 8 и профессиональные симуляторы экзамена, например, Whizlabs и Enthuware. Сегодня же у сдающих экзамен OCAJP8 такой проблемы нет. В этой статье вы познакомитесь с несколькими лучшими книгами для подготовки к экзамену OCAJP8.

Как подходить к сдаче экзамена

Правила успеха на экзамене OCAJP8 очень просты: найдите хороший учебник (например, любую из описанных ниже книг), пишите код каждый день, проходите как можно больше пробных экзаменов. Достаточно быть честным с самим собой при написании кода: не только писать, но и понимать, что вы пишете, и экспериментировать с кодом. При ответе на тренировочные вопросы из пробных экзаменов, внимательно изучайте все варианты ответов. Важно не только знать, какой ответ правильный, но и понимать, почему все остальные неправильны. И, наконец, обязательно читайте пояснения к тренировочным вопросам и пробным экзаменам. Это богатейший источник информации именно по данному экзамену. Кроме того, они помогут вам исправить свои ошибочные представления и заполнить пробелы в ваших знаниях. Когда я получил, еще во времена Java 1.4, свою первую сертификацию, именно эти три правила помогли мне добиться высокого балла, так что с тех пор я всегда рекомендую этот подход своим читателями. Никаких серебряных пуль – только честная подготовка с использованием правильной учебной литературы, книг и тренировочных вопросов.

Лучшие книги по OCAJP8

Хотя в платформе Java SE 8 появилось несколько новых концепций, например, лямбда-выражения, ссылки на методы, потоки данных, новые API для работы с датами и временем и др., больших отличий от OCAJP7 в ней не наблюдается. Так что если у вас уже есть какие-либо книги по OCAJP7, вы вполне можете воспользоваться ими для подготовки, а потом актуализировать свои познания в соответствии с требованиями нового экзамена. Если вы новичок или получаете свою первую сертификацию по языку Java, то можете использовать для подготовки к экзамену OCAJP8 с кодом 1Z0-808 следующие книги. Помните, что помимо различных версий Java (7 и 8) есть еще два разных экзамена – OCAJP8 и OCPJP8 – так что для выбора нужных книг обязательно нужно учитывать код экзамена.

OCA Java SE 8 Programmer I Certification Guide

Автора книги зовут Мала Гупта (Mala Gupta). Я с нетерпением ждал выхода этой книги с тех пор, как прочитал руководство по сертификации OCAJP7 того же автора. И она, безусловно, прекрасно дополнила его новыми возможностями Java 8. Вы найдете в этой книге массу полезных схем, заметок на полях, а также сжатое и понятное изложение различных граничных случаев, важных для получения сертификации по языку Java. Как всегда с подобными учебниками, эта книга охватывает все необходимые для экзамена OCA Java SE 8 Programmer I Certification (код экзамена 1Z0-808) вопросы, включая надстройки языка Java, например, создание и использование лямбда-выражений, новые API для работы с датами и временем и др. Она также включает полномасштабный пробный экзамен, который можно использовать вместе с сертификационными вопросами по Java 8 Дэвида Майера. Даже если вы хорошо знакомы с Java 8, уверен, что вы сможете найти в этой книге что-то новое для себя или узнать новые факты об уже знакомых вам вещах. В целом это отличная книга для Java-программистов всех уровней, готовящихся к сертификационному экзамену OCAJP8, так что, если хотите купить для подготовки всего одну книгу – покупайте именно её.

OCA: Oracle Certified Associate Java SE 8 Programmer I Study Guide: Exam 1Z0-808

Еще одна прекрасная книга для подготовки к экзамену OCAJP8. Это наиболее актуальное и полное изложение вопросов OCAJP8. Написали её Жанна Боярски (Jeanne Boyarsky) и Скотт Зеликофф (Scott Selikoff), проделавшие отличную работу по охвату всех важных нюансов экзамена OCAJP8. Книга начинается с теста для проверки своих знаний и содержит множество примеров для изучения как новых, так и старых принципов языка Java. Еще одна полезная часть этой книги – подробные пояснения к вопросам вышеупомянутого теста. Как и симулятор экзамена OCAJP 8 от компании Whizlabs, она не только показывает, почему ответ A или B – правильный, но и поясняет, почему остальные ответы ошибочны. Это очень полезно для понимания принципов работы и исправления своих ошибочных представлений.

OCA Java SE 8 Programmer I Study Guide (Exam 1Z0-808)

Это еще одна книга, посвященная экзамену OCAJP8, содержащая сведения из первоисточника, т. е. компании Oracle. 3-е издание этой, написанной Эдвардом Финеганом (Edward Finegan) и Робертом Лигуори (Robert Liguori), книги было полностью переработано и содержит актуальную и соответствующую темам экзамена информацию. Подобно предыдущей книге, она содержит множество упражнений и тестов для проверки своих знаний. К ней также прилагается компакт-диск, включающий два полномасштабных пробных экзамена. Хотя уровень вопросов отличается от реального экзамена, и лучше воспользоваться симулятором экзамена OCAJP 8 компании Whizlabs, тем не менее, на них можно неплохо попрактиковаться.

Oracle Java SE8 Programmer Exam Self-Practice Review Questions for Exam 1z0-808

Эта книга предназначена для отработки вопросов экзамена OCAJP8 (код 1Z0-808). Она содержит более 120 вопросов для тренировки. Её можно использовать, ради повышения качества подготовки, вместе с любой из вышеупомянутых книг. Эти тренировочные вопросы можно использовать вместе с моим списком бесплатных пробных экзаменов OCAJP8, чтобы набраться опыта перед тем, как перейти к финальной стадии подготовки в виде пробного экзамена на симуляторах экзаменов от компаний Enthuware или Whizlabls.
Вот и всё, что я хотел вам рассказать о нескольких неплохих книгах для подготовки к экзамену OCAJP8 (Сертифицированный разработчик Oracle на языке Java SE 8).
