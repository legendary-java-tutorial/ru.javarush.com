5 инструментов, которые улучшат вашу работу с GitHub

Источник: Medium GitHub – самый популярный сервис командной разработки. На нем можно найти бессчетное множество публичных репозиториев на различных языках программирования. Вы можете использовать GitHub, чтобы хранить свой код, работать с коллегами по команде и другими разработчиками. Также этот сервис можно использовать в учебных целях: изучать чужой код, популярные библиотеки с открытым кодом и принимать участие в их разработке. Кроме того, вы всегда можете усовершенствовать GitHub, «прокачав» и настроив его под себя. Представляем вам список инструментов, которые могут в этом помочь.

Awesome Autocomplete for GitHub

Одна из самых полезных фишек любого поиска – система автодополнения поискового запроса. И дело даже не в том, что так вы можете сэкономить время и не набирать весь свой запрос полностью. Главное, что тут вам предлагаются варианты запросов. Как вы уже догадались, при помощи расширения браузера Awesome Autocomplete for GitHub можно получить автодополнение в строке поиска на сайте GitHub. Его можно использовать для поиска как по репозиториям, так и по аккаунтам людей и организаций.

GitHunt

В функционале GitHub есть одна интересная возможность: там можно увидеть список трендовых проектов. То есть, вы можете ознакомиться с новыми инструментами и библиотеками, которые в последнее время привлекли к себе особое внимание и получили много звезд от других разработчиков. Чтобы облегчить процесс поиска новых трендов, используйте расширение браузера GitHunt. Оно позволяет просматривать трендовые проекты в новых вкладках по конкретному языку программирования за день, неделю или месяц.

GitHub Notable Comments

Один из самых полезных разделов GitHub, где можно найти рекомендации, советы или просьбы о помощи, относящиеся к конкретному проекту. В нем разработчики могут жаловаться на баги, предлагать решения или высказывать идеи о том, что, по их мнению, следует включить в проект. Если вы хотите быстрее перемещаться по комментариям и читать только самые полезные, воспользуйтесь расширением браузера GitHub Notable Comments. Установив его, зайдите во вкладку Issues нужного проекта и посмотрите на его содержимое. Для примера можно взять Issue проекта Rails. Там вы найдете 66 комментариев, но вам, естественно, хотелось бы прочесть самые полезные. Поможет в этом небольшая боковая панель, которая появляется после установки расширения. Так вы можете моментально перейти к комментарию, получившему больше всего тех или иных реакций.

Enhanced GitHub

Установив расширение Enhanced GitHub в своем браузере Google Chrome, вы сможете моментально получить ссылку на скачивание не только всего репозитория, но и любого файла в нем. Также у вас появится возможность копировать содержимое файлов. Хотя при копировании кода теряется разметка, но все равно это довольно полезный инструмент.

Git First

Если вы вдруг захотите посмотреть свой первый коммит в начатом вами проекте или первый коммит вашего коллеги, воспользуйтесь плагином Git First. Так вы существенно сократите время на поиски. Нужно лишь установить его в браузере, после чего у вас в контекстном меню (по правому клику мыши) появится дополнительный пункт: Initial Commit. Очень удобный инструмент, позволяющий быстро просматривать, как другие люди начинают свои проекты.
