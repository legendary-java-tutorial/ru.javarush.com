Java Tech Talk: Hand-made Spring Boot Starter

Spring Boot Starters — это набор дескрипторов зависимостей, которые вы можете подключить в свое приложение. Используя Spring Boot Starters, разработчик избавляет себя от необходимости копаться в примерах кода, копируя и вставляя множество дескрипторов зависимостей. Boot Starters экономит время на конфигурации и позволяет сосредоточиться непосредственно на проектировании приложения. Единственным недостатком этого инструмента является то, что иногда starter не предоставляет разработчику нужного ему функционала. Однако, и эту проблему вполне можно решить, ведь всегда можно создать собственный starter. Недавно прошедший стрим на канале Minsk Java Community посвящен этой задаче. Как реализовать с нуля свой Spring Boot Starter покажет и расскажет Тимофей Зубков.
