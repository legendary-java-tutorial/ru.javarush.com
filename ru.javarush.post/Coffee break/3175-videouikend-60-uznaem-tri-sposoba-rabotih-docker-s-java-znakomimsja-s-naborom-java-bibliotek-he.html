Три способа использовать DOCKER с JAVA

Уже потом вы можете запускать контейнер в любой необходимой вам среде: в том числе, на виртуальном сервере или в облаке. Существует, как минимум, три способа использования Docker для Java проекта. Дмитрий Корнилов — Кто такой Helidon и что с ним можно делать?

Дмитрий Корнилов, Java-разработчик из пражского офиса Oracle, представил доклад, посвященный Helidon — популярному набору Java-библиотек для создания микросервисов. Просмотрев видеоролик, вы узнаете, что такое Helidon, как он работает в системе современных средств разработки микросервисов и чем от них отличается. Часть доклада Дмитрия посвящена реактивному программированию, поддержке MicroProfile и GraalVM, современным средствам трейсинга и мониторинга. Кроме того, зрители увидят демо с построением двух RESTful сервисов с помощью реактивного программирования и MicroProfile.
