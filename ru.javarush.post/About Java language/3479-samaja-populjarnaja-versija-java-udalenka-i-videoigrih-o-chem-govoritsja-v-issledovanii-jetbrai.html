Самая популярная версия Java, удаленка и видеоигры: о чем говорится в исследовании JetBrains 2021 года

----------------------------------------

Компания-разработчик инструментов для Java и других языков JetBrains провела ежегодное исследование. В опросе Developer Ecosystem Survey 2021 года приняли участие более 47 тысяч...

----------------------------------------

Компания-разработчик инструментов для Java и других языков JetBrains провела ежегодное исследование. В опросе Developer Ecosystem Survey 2021 года приняли участие более 47 тысяч человек из 183 стран. В том числе, во время исследования опросили айтишников из 17 стран, на которые приходится примерно 70% всех разработчиков в мире: Аргентина, Беларусь, Бразилия, Канада, Китай, Франция, Германия, Индия, Япония, Мексика, Россия, Южная Корея, Испания, Турция, Украина, Великобритания и США. Мы собрали ключевые выводы исследования — про язык Java, работу и образ жизни программистов.

Java в 2021 году: популярна в Южной Корее, Китае и Германии

Одно из главных заключений исследования — это то, что язык программирования Java хоть и немного теряет позиции, все же более популярен, чем Python в качестве основного языка (32% против 29% опрошенных). Самым популярным языком в 2021 году стал JavaScript — 69% респондентов использовали его за последние 12 месяцев, а 39% назвали его основным языком программирования. Как и в 2020 году, Java чаще всего используют в Южной Корее, Китае и Германии. Доля Java в Южной Корее составляет 55%, в Китае — 47%, а в Германии — 33%. В Беларуси 26% разработчиков предпочитают Java, в Украине — 20%, в России — 18%. Это немного меньше, чем в 2020 году.

Какая версия Java самая популярная?

Как и в 2020 году, Java 8 по-прежнему ведущая версия этого языка программирования. Интересно и то, что начиная с 2019 года Java 11 ежегодно демонстрирует стабильный рост примерно на 10 процентных пунктов. В цифрах это выглядит так: 72% разработчиков, которые используют Java, отдают предпочтение Java 8, а 42% — Java 11. Доли Java 12 и Java 13 снизились примерно в три раза с 2020 года, в то время как более новые Java 14 и Java 15 уже получили аудиторию несмотря на то, что они относительно новые.

Какие фреймворки выбирают

Доля разработчиков, использующих Spring Boot, немного выросла в 2021 году, примерно на 4 процентных пункта (его выбирают 65% программистов). На втором месте фреймворк Spring MVC — 42% разработчиков. В это же время небольшая часть разработчиков использует гораздо больше новых веб-фреймворков, чем в 2020 году.

Серверы приложений: Apache Tomcat и Jetty

Ландшафт серверов приложений в 2021 году примерно такой же, как и в 2020 году, Apache Tomcat по-прежнему наиболее популярный среди разработчиков — его используют 61% программистов.

Для чего чаще всего используют Java?

Чаще всего Java используют для разработки вебсайтов (39%), реже — для создания утилит (26%) и системного ПО (19%). Использование Java для веб-сайтов и ИТ-инфраструктуры незначительно выросло в период с 2020 по 2021 год, в то время как Java стала меньше использоваться для бизнес-аналитики, обработки данных и машинного обучения.

Портрет разработчика: в отрасль приходит больше женщин

63% девелоперов работают на фултайме, 15% — учатся в вузах, 7% — также учатся, но работают, 5% — фрилансят. 81% респондентов работают на позиции программиста, 15% — тимлида, 13% — DevOps-инженера, 12% — архитектора. Большинство программистов работают в компаниях, где разработчиков меньше, чем специалистов других областей. Еще один примечательный вывод — это то, что чаще всего разработчики трудятся в компаниях численностью не больше 50 человек (так ответили 80% опрошенных). Если говорить об IT-компаниях, то чаще всего программисты работают с IT-сервисами (40%), мобильной разработкой (24%), облачными платформами и данными (по 17%). Среди компаний из других сфер айтишники чаще всего трудятся в сфере банкинга и финансов (18%), продаж и образования (по 12%). Основную долю среди разработчиков занимают мужчины (93%) от 21 до 29 лет (48%). Тем не менее, все больше женщин приходят в технологическую отрасль. Они чаще работают с анализом данных и машинным обучением, UX/UI-дизайном. Гораздо меньше женщин работают с программированием и системным администрированием. Самый большой процент программисток в Южной Корее (13%), Беларуси (12%) и Канаде (8%). В России в общем поле разработчиков только 7% программисток, а в Украине — 6%.

Образ жизни разработчика: время пробуждения и удаленка

Время, в которое разработчики обычно начинают свой рабочий день, не изменилось с прошлого года: программисты придерживались своего рабочего графика даже во время пандемии. 83% разработчиков начинают свой рабочий день до 10 часов утра. Если до пандемии около 70% разработчиков работали из офиса, то сейчас 80% работают из дома. Интересно, что хотя во время пандемии доля разработчиков, работающих в кафе, сократилась вдвое, количество людей, работающих в коворкингах, не сильно изменилось. В этом году разнообразные мероприятия стали намного доступнее благодаря широкому распространению онлайн-форматов. В 2020 году около 75% разработчиков посетили онлайн-конференции, а более 40% респондентов посетили онлайн-семинары и встречи. Хотя у абсолютного большинства разработчиков есть учетная запись GitHub, чуть меньше половины имеют профиль на Stack Overflow. Три наиболее часто используемых социальных сети — это Twitter, Instagram и Facebook.

Хобби разработчиков: видеоигры опережают кодинг

В 2020 году большинство разработчиков назвали программирование своим самым большим хобби. В этом году видеоигры возглавляют список, их популярность выросла с 49% до 59%. Персональный компьютер — самая популярная игровая платформа, консоли заняли второе место. Среди них наиболее широко используются PlayStation и Nintendo Switch. На втором месте по популярности среди хобби — программирование (57%), на третьем — прослушивание музыки (40%). Более половины опрошенных занимаются благотворительностью. Самая популярная причина благотворительных пожертвований — это желание сделать мир и сообщество более удобными для жизни. Большинство разработчиков желают помочь детям, заботятся об окружающей среде, борются с социальным неравенством или помогают обездоленным. Таковы итоги исследования экосистемы разработки в 2021 году.
