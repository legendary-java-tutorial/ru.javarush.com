Рейтинг TIOBE: Java по-прежнему на первом месте

----------------------------------------

Согласно индексу TIOBE, оценивающему популярность языков программирования на основе подсчёта результатов поисковых запросов, в сентябре Java по-прежнему удерживает пальму первенства. Результат нашего с вами любимого языка — 12.6% от общего количества запросов.

Согласно индексу TIOBE, оценивающему популярность языков программирования на основе подсчёта результатов поисковых запросов, в сентябре Java по-прежнему удерживает пальму первенства. Результат нашего с вами любимого языка — 12.6% от общего количества запросов. Более, чем на 5% отстаёт ближайший преследователь — «старейшина» программистского мира Си, а его младший по возрасту, но очень мощный объектно-ориентированный братишка, С++, закрепился на третьем месте. Пятёрку лидеров замыкают C# и Python. Последнему уже который год пророчат скорое лидерство, но не все аналитики согласны с этим мнением.
