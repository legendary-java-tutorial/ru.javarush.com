1. Установка плагина

Чтобы значительно упростить вам жизнь, в JavaRush написали специальный плагин для IntelliJ IDEA, который позволит вам получать и отправлять задачи на проверку за пару кликов. Это очень удобно – сейчас вы сами во всем убедитесь.

Плагин JavaRush пока не входит в стандартный набор плагинов в репозитории IntelliJ IDEA, поэтому вам придется скачать его с нашего сайта и установить в IDEA вручную. Сделать это довольно просто.

Перед установкой нужно «Скачать плагин».

ШАГ 1. Запустите IntelliJ IDEA и перейдите в меню File -> Settings. Для MacOS в меню нажмите IntelliJ IDEA -> Preferences.

ШАГ 2. Найдите пункт Plugins. Нажмите на шестеренку и выберите Install Plugins from disk.

ШАГ 3. Откройте папку, в которую скачали плагин (плагин — это файл JavaRushIdeaPlugin.jar), выберите его и нажмите OK.

ШАГ 4. Перезапустите IntelliJ IDEA (Restart).

ШАГ 5. После того, как IDEA перезагрузилась и плагин JavaRush активировался, в верхнем ряде кнопок IDEA добавится новая группа из семи кнопок.

ШАГ 6. Создание проект для задач JavaRush. Нажмите на любую кнопку плагина. Плагин сам определит, что проект JavaRush еще не загружен и предложит скачать его.

ШАГ 7. Выберите папку для загрузки проекта. Не используйте системные папки (Program Files и тп).

ШАГ 8. После скачивания IDEA сама откроет его, и вы увидите инструкцию вроде такой:

ШАГ 9. Дополнительные настройки проекта. Жмем File -> Project Structure.

Шаг 10. В разделе Project SDK выберите версию Java 13 или 14. В поле Project language level выберите версию 13 - No new language features: это важно.

Шаг 11. Если пишет No SDK. Жмем Add SDK -> выбираем папку с Open JDK. На этом все. Можете приступить непосредственно к решению задач.

2. Решение задач в плагине JavaRush

Решать задачи с помощью плагина — одно удовольствие. Для начала вам нужно выбрать одну из доступных вам задач. Вот что вам следует делать:

ШАГ 1. Открой список доступных задач нажав на кнопку:

ШАГ 2. Вы должны увидеть список доступных задач. Если список пуст, значит вы решили все доступные задачи: вам нужно открывать новые лекции, чтобы получить больше задач. Если задачи в списке есть, просто кликайте по понравившейся задаче.

ШАГ 3. Перед вами открылось окно с двумя вкладками. Одна — с условием, вторая (Solution) — для ввода кода. Всё! Вы готовы решать задачу.

ШАГ 4. Перейдите во вкладку Solution и напишите решение задачи. Выглядеть оно может примерно так:

ШАГ 5. Если вы считаете, что решили задачу правильно, настало время отправить ее на проверку. Для этого жмите на «кнопку с галочкой»:

ШАГ 6. Если задача не прошла тестирование, вы увидите причины, а также список рекомендаций. Если прошла — поздравляю! Вы честно заработали награду — чёрную материю.
