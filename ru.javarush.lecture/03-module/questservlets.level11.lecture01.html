Загрузка Tomcat

Ты можешь установить Tomcat к себе на компьютер двумя способами: с помощью Windows Installer или скачав его в виде архива. Первый способ проще, второй же позволяет управлять всеми аспектами настройки и установки веб-сервера.

Давай начнем с простого способа, а дальше ты уже решишь для себя, какой тебе больше по душе.

Скачать Tomcat можно с его официальной страницы. На этой странице мы можем увидеть различные опции для загрузки: в виде архива, который достаточно распаковать, либо в виде инсталлятора.

Можно установить Tomcat разными способами, но мы для простоты выберем пункт 32-bit/64-bit Windows Service Installer.

Запуск и остановка сервиса

Если тебе нужно запустить или остановить сервис Tomcat в Windows, то для этого нужно:

Шаг 1. Открыть диспетчер задач Ctrl+Shift+Esc.

Шаг 2. Открыть вкладку Services/Службы и найти там сервис Tomcat.

Шаг 3. Останавливаем сервис Tomcat9.

В правой колонке отображается его состояние Running/Выполняется или Stopped/Остановлено.

Кликаем правой кнопкой мышки на строке Tomcat9 и жмем Stop Service/Остановить.

Для остановки сервиса нужно нажать Stop Service, этот пункт будет активен, если сервис Tomcat запущен.
