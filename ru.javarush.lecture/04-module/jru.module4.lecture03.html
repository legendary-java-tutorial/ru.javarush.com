Задание - создать 6 таблиц в созданных ранее трех схемах

----------------------------------------

Задание: Следуя инструкциям из шестого урока пятого уровня:

  1. Создай в схеме space таблицу object с пятью колонками:
    • id типа INT
    • name типа VARCHAR(100)
    • type типа VARCHAR(50)
    • discoverer типа VARCHAR(100)
    • discovery_date типа DATE
  2. создай в схеме library:
    1. таблицу publisher с двумя колонками:
      • id типа INT
      • name типа VARCHAR(100)
    2. таблицу author с четырьмя колонками:
      • id типа INT
      • first_name типа VARCHAR(50)
      • last_name типа VARCHAR(50)
      • full_name типа VARCHAR(100)
    3. таблицу book с шестью колонками:
      • id типа INT
      • title типа VARCHAR(100)
      • author_id типа INT
      • publisher_id типа INT
      • publication_year типа INT
      • isbn типа BIGINT
  3. Создай в схеме lego:
    1. таблицу part с четырьмя колонками:
      • id типа INT
      • number типа VARCHAR(25)
      • name типа VARCHAR(100)
      • weight типа FLOAT
    2. таблицу lego_set с пятью колонками:
      • id типа INT
      • number типа INT
      • name типа VARCHAR(100)
      • released типа INT
      • inventory типа INT
