Java @Аннотации. Что это и как этим пользоваться?

Аннотации в Java — это своего рода метки в коде, которые описывают метаданные для функции/класса/пакета. Тема сложновата для новичков, но для твоего нынешнего базиса знаний — в самый раз.

Данная статья предназначена для людей, которые никогда не работали с Аннотациями, но хотели бы разобраться, что это и с чем его едят.

Минисерия об аннотациях

  • Аннотации. Часть первая, немного скучная. Здесь коротко описаны аннотации с типом SOURCE и CLASS. Стоит прочитать, чтобы не потеряться во второй части и чтобы начать "не понимать" немного больше :)
  • Аннотации. Часть вторая. Lombok. В этой части рассказывается о библиотеке Lombok — известном представителе Source аннотаций
